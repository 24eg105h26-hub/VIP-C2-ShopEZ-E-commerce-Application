1. Introduction
This document presents the Data Flow Diagrams (DFDs) and formal User Stories for
the  ShopEZ  platform.  DFDs  model  the  system  at  two  abstraction  levels  —  Context
Level  (Level  0)  and  Functional  Decomposition  (Level  1)  —  illustrating  data  flows
between actors, processes, and data stores.
